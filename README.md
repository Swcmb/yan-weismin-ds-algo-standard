## 概述

本技能为 AI 提供了一套严格遵循严蔚敏《数据结构》教材风格的类 C 语言算法书写规范。在编写数据结构基本操作、教材课后习题算法、或类 C 语言伪代码时，AI 将自动遵循本规范，确保代码风格与教材一致。

## 适用场景

- 编写数据结构基本操作算法（顺序表、链表、栈、队列、串、树、图、查找、排序等）
- 完成严蔚敏《数据结构》教材课后习题
- 使用类 C 语言描述算法或书写伪代码
- 按教材风格进行算法教学与演示

## 核心内容

### 一、类 C 语言语法概要

| 类别 | 要点 |
|---|---|
| **预定义常量** | `TRUE`/`FALSE`、`OK`/`ERROR`/`INFEASIBLE`/`OVERFLOW` |
| **预定义类型** | `Status`（返回状态）、`bool`（布尔类型）、`ElemType`（数据元素类型） |
| **数据结构表示** | 用 `typedef` 描述存储结构 |
| **函数形式** | 函数类型 + 函数名 + 参数表 + 算法说明注释 |
| **引用参数** | 以 `&` 打头的引用调用（C++ 风格） |
| **赋值语句** | 简单/串联/成组/结构/数组/数组段/交换/条件赋值 |
| **选择语句** | `if`、`if-else`、`switch`（值）、`switch`（条件） |
| **循环语句** | `for`、`while`、`do-while` |
| **结束语句** | `return`、`break`、`exit` |
| **基本函数** | `max`、`min`、`abs`、`floor`、`ceil`、`eof`、`eoln` |
| **逻辑运算** | `&&` 和 `||` 短路求值 |

### 二、算法书写规范

1. **算法说明**（规格说明）—— 函数首部之下以注释形式写明功能、参数含义、限制条件等
2. **注释与断言**—— 关键处加注释，提倡断言式注释，书写入口断言
3. **输入输出**—— 优先使用参数传递，避免全局变量
4. **错误处理**—— 用 `Status` 返回状态代码
5. **语句选用**—— 基本三语句（赋值、if、while）+ switch，禁用 goto
6. **基本运算**—— 不得直接引用教科书基本运算，非用不可需同时实现
7. **边界验证**—— 用边界条件验证算法正确性
